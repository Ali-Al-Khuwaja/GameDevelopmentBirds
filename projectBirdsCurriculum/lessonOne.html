Introduction to Godot Engine Editor

The editor interface or the Godot engine interface is considered very easy, do not panic, it is an interface full of icons and simple terms that describe its function to a seven-year-old child.Most likely, you downloaded the engine, ran the program, and tried tinkering here and there, and you did not understand much of what happened. Well, let us take a short trip to the editor to understand what we see in front of us.


"Derived from the engine documentation"

Projects Manager window

When you launch Godot, the first window you see is the Project Manager. This window allows you to create, remove, import, or run game projects.

In the upper right corner of the window, the drop-down menu allows you to change the language of the editor. You can change the language of the engine to any if you want, but here in projectBirds we will use the English language since the engine has not been fully translated to other languages yet.

Creating and importing projects

To create a new project, click the New button at the top left of the Project Manager window , Name the project then choose an empty folder on your computer to save the files, then select the Renderer you want. Click the Create & Edit button.To create a project folder and open it in a new editor

Browsing files

Click the Browse button next to the x to open Godot's file browser and choose a location or type the folder path in the Project Path field.

Opening and importing projects

The next time you open the Project Manager, you'll see your new project in the list. Double click on it to open it in the editor.

You can likewise import existing projects using the "import" button. Locate the folder containing the project or project.godot file to import and edit.

When the folder path is correct, you'll see a green check mark.
